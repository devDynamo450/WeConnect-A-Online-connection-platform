# 🌐 WeConnect

> A real-time full-stack social communication platform combining social feeds, messaging, notifications, user profiles, and a friend system.

## 🚀 Tech Stack

| Layer     | Technology                                   |
|-----------|----------------------------------------------|
| Frontend  | HTML5, CSS3, TypeScript, Axios, Socket.IO    |
| Backend   | Node.js, Express.js, TypeScript, Socket.IO   |
| Database  | MongoDB + Mongoose                            |
| Auth      | JWT + bcrypt                                  |
| Testing   | Jest + Supertest                              |

## 📂 Project Structure

```
WeConnect/
├── backend/        # Express + TypeScript API server
├── frontend/       # HTML + CSS + TypeScript client
└── README.md
```

## 🧭 Development Phases

- **Phase 1** — Foundation (Backend setup, TypeScript, MongoDB)
- **Phase 2** — Authentication (Register, Login, JWT, Protected Routes)
- **Phase 3** — Social Feed (Posts, Likes, Comments)
- **Phase 4** — Messaging System (Socket.IO, Real-time chat)
- **Phase 5** — UI & Responsive Design
- **Phase 6** — Testing & Deployment

## ▶️ Getting Started

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (in a separate terminal)
cd frontend
npm install
npm run dev
```
